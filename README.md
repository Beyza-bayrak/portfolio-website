# Personal Website

Bu proje, modern bir kişisel web sitesi tasarımıdır.  
Web sitenizi kendiniz özelleştirebilirsiniz; boş alanları düzenleyerek kendi bilgilerinizi ekleyebilirsiniz.

## Özellikler
- Ana sayfa, hakkında, eğitim, sertifikalar, yetenekler ve iletişim bölümleri
- Temiz ve kullanıcı dostu arayüz
- Kolay düzenlenebilir yapı

## Kurulum
```bash
npm install
npm run dev
